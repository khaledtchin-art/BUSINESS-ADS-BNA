FAQ - Foire aux questions

Comment créer un compte ?
Rendez-vous sur la page d'inscription, remplissez le formulaire (nom, email, WhatsApp, pays) et effectuez le paiement de 1 400 FCFA.
Comment fonctionne le parrainage ?
Parrainez quelqu'un avec votre code unique. Quand il s'inscrit, vous gagnez 500 FCFA (niveau 1), puis 200, 100, 50, 25 FCFA pour les niveaux suivants. Total possible : 875 FCFA par filleul direct.
Comment retirer mon argent ?
Allez dans "Retrait", entrez le montant (minimum 10 000 FCFA), fournissez votre numéro Wave. Les frais sont de 2%. KYC obligatoire. Délai : 24-48h.
Comment déposer de l'argent ?
Allez dans "Dépôt", choisissez votre montant (minimum 5 000 FCFA), sélectionnez votre moyen de paiement (Wave, Orange Money, Moov Money).
C'est quoi le coffre-fort ?
Le coffre-fort vous permet de bloquer votre argent pour générer des intérêts. Verrouillage minimum 30 jours = 5% d'intérêts par mois.
Comment acheter sur Chariow ?
Allez dans "Offres", choisissez votre produit, cliquez sur "Commander". Paiement sécurisé par Wave, Orange Money, Moov Money ou carte bancaire.
C'est quoi le KYC ?
KYC = "Know Your Customer". C'est une vérification d'identité obligatoire pour retirer de l'argent. Vous devez fournir une pièce d'identité (CNI ou passeport).
